The most common text formatting tags in HTML.

  1. Heading tags: Used to define headings or subheadings on a web page.
    • Use <h1> for the main heading of the page.
    • Use <h2> for subheadings.
    • Use <h3> for sub-subheadings, and so on.
    • Don't skip heading levels (e.g., don't use an <h3> without an <h2>).
    • Use heading tags to create a clear hierarchy of content on your page.
  2. Paragraph tag: Used to define paragraphs of text on a web page.
    • Use <p> to wrap blocks of text that make up a paragraph.
    • Don't use <p> for headings or subheadings.
    • Avoid using <br> tags to create line breaks within paragraphs. Instead, use CSS to control spacing and margins.
  3. Strong and Emphasis tags: Used to add emphasis or importance to text on a web page.
    • Use <strong> to indicate strong emphasis.
    • Use <em> to indicate emphasis or to highlight words or phrases.
    • Don't use these tags for styling purposes. Use CSS for styling.
  4. Anchor tag: Used to create links to other web pages or resources.
    • Use <a> with the href attribute to specify the URL of the page or resource you're linking to.
    • Use descriptive text for the link (i.e., avoid using "click here").
    • Use the target="_blank" attribute to open the linked page in a new browser tab or window.
  5. List tags: Used to create ordered and unordered lists.
    • Use <ul> for unordered lists (bullet points).
    • Use <ol> for ordered lists (numbered items).
    • Use <li> to define each list item.
    • Use nested lists to create sub-items or sub-categories within your lists.
